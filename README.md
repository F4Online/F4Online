# F4Online

F4Online is a multiplayer framework for Fallout 4 focused on bringing players together in a shared wasteland experience.

The project aims to provide seamless multiplayer integration directly within Fallout 4 while staying as close as possible to the base game. Players can host or join sessions through an in-game interface and explore the Commonwealth alongside others without requiring major gameplay overhauls.

## Goals

- Seamless multiplayer integration
- In-game server hosting and joining
- Support for public and private sessions
- Up to 32 players per server
- Lightweight and easy to install
- Minimal dependencies
- Focus on stability and compatibility

## Current Vision

F4Online is designed around a simple idea: let players experience Fallout 4 together.

The project prioritizes player synchronization, exploration, roleplay, social interaction, and cooperative gameplay while preserving the look and feel of the original game. The goal is to make multiplayer feel like a natural extension of Fallout 4 rather than a separate experience.

## Status

F4Online is currently in active development.

Features, implementation details, and design decisions may change as development continues.
